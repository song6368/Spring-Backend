# 게시판 RestAPI 백엔드

## 소개
이 프로젝트는 게시판 REST API 백엔드의 기본 기능 구현을 목표로 개발되었습니다.

## 목표 기능
- 회원가입/로그인 ( 완료 )
- 게시글 CRUD ( 완료 )
- JWT ( 완료 )
- 사용자 인증 AOP 적용 ( 완료 )
- 댓글 ( 완료 )
- 대댓글 ( 완료 )
- 게시글 및 댓글 추천/비추천
- 웹 소켓활용 채팅 시스템

## 기술 스택
- Java
- Spring Boot
- Spring Data JPA
- JWT (JSON Web Token)
- PostgreSQL (데이터베이스)
